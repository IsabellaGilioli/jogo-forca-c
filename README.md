# 🎮 Jogo da Forca em C

Projeto desenvolvido em linguagem C durante estudos de programação.  
O jogo simula a clássica **forca**, com leitura de palavras de arquivo e interação via terminal.

---

## 🚀 Funcionalidades

- Escolha aleatória de palavras
- Sistema de chutes de letras
- Conversão automática para maiúsculas
- Contagem de erros
- Desenho da forca no terminal
- Permite adicionar novas palavras ao jogo
- Uso de arquivos (`palavras.txt`)

---

## 🛠️ Tecnologias usadas

- Linguagem C
- Manipulação de arquivos (`fopen`, `fscanf`, `fprintf`)
- Strings e arrays
- Biblioteca `ctype.h`
- Controle de fluxo e funções

---

## ▶️ Como compilar e executar

No terminal:

```bash
gcc forca.c -o forca
./forca
